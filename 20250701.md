-------------------------------------------
テッペン探し  
・地図が正方形で、その横縦の長さが与えられる  
・地図に各マスに標高が1つずつ与えられるされ、それぞれの標高が与えられる  
・上下左右の山より標高が高い場合、調査対象の山をテッペンだと判断  
・地図の範囲外は標高0扱い。  
・1地図に1テッペンは必ずある  
・テッペンを高〜低順に出力  
 
-------------------------------------------
 
```ruby
len = gets.to_i                                                     # 地図の横縦の長さを取得  
plan = len.times.map{gets.split.map(&:to_i)}                        # 全部の山の標高を取得
height = []                                                         # テッペンを記録
plan.each_with_index do |h,x|                                       # 各山を確認していく
  h.each_with_index do |w,y|                                        # 上下左右の山を定義して確認
    top = x - 1 < 0 ?  0 : plan[x-1][y]
    down = x + 1 < len ? plan[x+1][y] : 0
    left = y - 1 < 0 ? 0 : plan[x][y-1]
    right = y + 1 < len ? plan[x][y+1] : 0
    height << plan[x][y] if plan[x][y] > [top,down,left,right].max  # 調査対象の山が上下左右の山より標高が高い場合、調査対象の山をテッペンだと判断して記録  
  end
end

puts height.sort.reverse                                            # テッペンを高〜低順に出力
```

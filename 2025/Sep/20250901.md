Bランク　1問  
 
-------------------------------------------
スケジュール管理  
・登壇者数が与えられる  
・それぞれの名前、持ち時間(分)が与えられる  
・開始時間は10:00〜  
・登壇者の交代時に、10分休憩を挟む  
・前の登壇者の終了時間+10分休憩後の時間が12:00を過ぎた場合、次の登壇を一旦stopして1時間の昼休憩  
・各登壇者の  開始時間 - 終了時間  名前　を出力
 
-------------------------------------------
 
```ruby
amounts = gets.to_i                                                                            # 登壇者数を取得
people = []                                                                                    # 各自の情報を記録用
amounts.times do
  name, time = gets.split                                                                      # 各登壇者の名前と時間を記録
  people << [name,time.to_i]
end

st = [10,0]                                                                                    # 開始時間を初期化
fin = []                                                                                       # 終了時間を記録用
sched = []                                                                                     # 登壇した人を記録用

people.each do |name,mins|                                                                     # 各登壇者の時間表を作成
break if st[0]+(st[1]+mins)/60 >=12 && (st[1]+mins)%60 >=1                                     # 昼休憩の時間を超えそうだったら処理終了
  fin = [(st[0]+(st[1]+mins)/60)%24,(st[1]+mins)%60]                                           # 終了時間を定義
  puts "#{st[0]}:#{sprintf("%02d", st[1])} - #{fin[0]}:#{sprintf("%02d", fin[1])} #{name}"     # 登壇者の時間表を出力
  st = [fin[0]+(fin[1]+10)/60, (fin[1]+10)%60]                                                 # 休憩挟んだ後の時間を次の登壇者の開始時間に設定
  sched << [name,mins]                                                                         # 登壇した人の情報を記録
end

st = [fin[0]+1,fin[1]]                                                                         # 昼休憩の1時間加算

people.each do |name,mins|                                                                     # 昼休憩後の時間表を作成
  next if sched.include?([name,mins])                                                          # 登壇した人をskip
  fin = [(st[0]+(st[1]+mins)/60)%24,(st[1]+mins)%60]                                           # 終了時間を定義
  puts "#{st[0]}:#{sprintf("%02d", st[1])} - #{fin[0]}:#{sprintf("%02d", fin[1])} #{name}"     # 登壇者の時間表を出力
  st = [fin[0]+(fin[1]+10)/60, (fin[1]+10)%60]                                                 # 休憩挟んだ後の時間を次の登壇者の開始時間に設定
end
```

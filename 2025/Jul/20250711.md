スキルチェック  
Bランク　1問  
 
-------------------------------------------
最大Xを探す  
・方眼紙の縦横が与えられる  
・各マスに「.」か「#」が記入されている  
・「.」のマスを中心に、↖️↙️↗️↘️斜め4方向に全部「.」が続いているなら、「X」が描けるとする  
・中心からの拡張距離を記録し、拡張できる最大マス数を出力  
・「X」が描けない場合は 0 を出力  
 
-------------------------------------------
 
```ruby
height, width = gets.split.map(&:to_i)                                               # 方眼紙の縦横を取得
papper = height.times.map{gets.chomp.chars}                                          # 方眼紙の各マスの内容を取得
dots = []                                                                            # 「.」が記入されてるマスの座標を記録用

papper.each_with_index {|h, x|h.each_with_index {|w, y| dots << [x,y] if w == "."}}  # 「.」が記入されてるマスの座標を取得

max = 0                                                                              # 最大拡張マス数を記録用
dots.each do |x,y|                                                                   # 「.」が記入されてるマスを確認していく
ext = 1                                                                              # 拡張マス数を1から設定
  loop do                                                                            # 同じ座標で最大拡張数を試していく
    lt = papper[x-ext][y-ext] if x-ext>=0 && y-ext>=0                                # ↖️↙️↗️↘️4方向のマスを確認
    ld = papper[x+ext][y-ext] if x+ext<height && y-ext>=0
    rt = papper[x-ext][y+ext] if x-ext>=0 && y+ext<width
    rd = papper[x+ext][y+ext] if x+ext<height && y+ext<width
    diagonal = [lt,ld,rt,rd] if [lt,ld,rt,rd].count(".") == 4                        # 4方向全部.の場合記録、1つでも欠けたらnilになる
      if diagonal == nil                                                             # 欠けがあった場合
         max = [max,ext-1].max                                                       # 現座標の拡張マス数が全座標の最大拡張マス数を超えた場合のみ上書き
         break                                                                       # 現座標の処理を終了
      end
    ext += 1                                                                         # マスを1つ拡張
   end
end

puts max                                                                             # 最大拡張マス数を出力
```

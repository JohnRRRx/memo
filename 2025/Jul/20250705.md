ポイントしか使わなかったら...？  
・1週間限定のキャンペーンで、ポイントを使うと、一定数が還元される  
・所持ポイントとポイント還元率が与えられる  
・毎日1回ポイント全額使用  
・最終的に合計使用ポイントを出力  
  
-------------------------------------------
  
```ruby
init = gets.to_i           # 所持ポイントを取得
rt = gets.to_i             # 還元率を取得
used = 0                   # 使用したポイントを記録用

7.times do                 # 1週間分の使用ポイントを集計
 used += init              # 使用したポイントを累計
 init = init * rt / 100    # 還元されたポイントを次の日の使用分に変換
end

puts used                  # 合計使用ポイントを出力
```

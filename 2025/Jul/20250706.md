-------------------------------------------
迷路探索  
・マス付きの地図の縦横の長さが与えられる  
・開始地点の座標が与えられる  
・下記記号で各マスの情報が与えられる  
・下記ルールを上から優先順で進む  
・終点の扉座標を出力  

| 記号 | 意味 |
|:-----------:|:------------:|
| . | 道 |
| # | 壁 |


| 🧭 | 🦶 |
|:-----------:|:------------:|
| 西 | 道があれば突き当たりまで移動し、南1マス |
| 東 | 道があれば突き当たりまで移動し、南1マス |
| 南 | 東西とも壁の場合、南1マス |
 
-------------------------------------------
 
```ruby
height,width = gets.split.map(&:to_i)                    # 地図の縦横の長さを取得
ent_x,ent_y = gets.split.map(&:to_i)                     # 開始地点の座標を取得
a = ent_x-1                                              # 開始地点の座標をindex化
b = ent_y-1
maze = height.times.map{gets.chomp.chars}                # 各マスの情報を取得

loop do
  if b > 0 && maze[a][b - 1] == "."                      # 西に道があれば
    b -= 1 while b > 0 && maze[a][b - 1] == "."          # 突き当たりまで移動
    a += 1 if a + 1 < height && maze[a + 1][b] == "."    # 南に1マス移動

  elsif b < width - 1 && maze[a][b + 1] == "."           # 東に道があれば
    b += 1 while b < width - 1 && maze[a][b + 1] == "."  # 突き当たりまで移動
    a += 1 if a + 1 < height && maze[a + 1][b] == "."    # 南に1マス移動

  elsif a + 1 < height && maze[a + 1][b] == "."          # 南に道があれば1マス移動
    a += 1

  else
    break                                                # どれも当てはまらない場合、終点だと判断して処理終了
  end
end

puts "#{a + 1} #{b + 1}"                                 # 終点の扉座標を出力
```

-------------------------------------------
表作成  
・項目の数が与えられる  
・項目が与えられる  
・データの数が与えられる  
・データが与えられる  
・下記例のように、項目名→区切り線→値の順で出力
・同じ縦列で、最大文字数に応じて半角spaceを足す
・区切り線は特にspaceは入れない  
  
```
例：    
| key1   | key2   | key3   |
|--------|--------|--------|
| value1 | value2 | value3 |
| value4 | value5 | value6 |
| value7 | value8 | value9 |
```
-------------------------------------------
  
```ruby
e_amounts = gets.to_i                                     # 項目の数を取得
elements = gets.chomp.split                               # 項目を取得
d_amounts = gets.to_i                                     # データの数を取得
datas = d_amounts.times.map {gets.chomp.split}            # データを取得

turn = datas.transpose                                    # データを縦横変換

sets = {}                                                 # 縦各列の最大文字数を記録用
elements.each_with_index do |e, idx|                      # 縦各列の文字数を確認してく
  lengths = [e.size] + turn[idx].map(&:size)
  sets[e] = lengths.max                                   # 縦各列の最大文字数をHash化
end

s_arr = sets.to_a                                         # HashをArray化

titles = []                                               # 項目名を記録用
elements.each_with_index do |e, idx|                      # 項目名の文字数を確認してく
  width = s_arr[idx][1]
  dif = " " * (width - e.size)                            # 最大文字数との差を算出し、差分の半角spaceを作成
  titles << " #{e}#{dif} "                                # 結果を項目名に記録
end
puts "|#{titles.join('|')}|"                              # 項目名の行を整形して出力

puts "|#{s_arr.map { |_, w| '-' * (w + 2) }.join('|')}|"  # 区切り線を列ごとの幅に合わせて出力

datas.each do |row|                                       # データを確認してく
  line = row.each_with_index.map do |val, idx|            # 縦各列の値を処理
    width = s_arr[idx][1]                                 # 現在の列の最大文字数を取得
    dif = " " * (width - val.size)                        # 最大文字数との差を算出し、差分の半角spaceを作成
    " #{val}#{dif} "                                      # 値と半角spaceを合わせて整形
  end
  puts "|#{line.join('|')}|"                              # 整形したデータの行を出力
end
```

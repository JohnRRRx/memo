Cランク　1問  
  
-------------------------------------------
+1  
・英語の大文字と数字で組み合わせた文字列(〜9文字)が与えられる  
・英文字の部分を1つ後の文字に変更  
・数字は+1に変更、2桁以上の数字が連続した場合、１つの数字として見なす  
・変更後の文字列を出力  

例：
| before | after |
|:-----------:|:------------:|
| ABC555 | BCD556 |
| 99WXY | 100XYZ |
| ZZ66XX | AA67ZZ |
  
-------------------------------------------
  
```
<?php
    $target = trim(fgets(STDIN));                                               # 文字列を取得
    $a2z = range("A","Z");                                                      # index判定用、A〜Z配列作成
    $parts = preg_split('/(\d+)/', $target, -1, PREG_SPLIT_DELIM_CAPTURE);      # 正規表現で数字を区切りとして文字列を分割(空要素も入ってしまう)
    $result = [];                                                               # 結果を記録用
    foreach ($parts as $part) {                                                 # 各部分を処理していく
      if (is_numeric($part)){                                                   # 数字の場合
          $result[] = $part+1;                                                  # １つの数字として記録
      } else {                                                                  # それ以外の場合(文字の場合)
        $changed = [];                                                          # 変更後の文字を記録用
        $part = array_filter(str_split($part));                                 # 空要素を排除
        foreach ($part as $char){                                               # 文字列を1文字ずつに分割
          $idx = array_search($char,$a2z);                                      # 1文字ずつのindex番号を取得
          $changed[] = $a2z[($idx+1)%26];                                       # 1つ後ろの英文字に変更(26文字あるので、%26でZ→Aに戻してループ可能)
        }
        $result[] = implode($changed);                                          # 変更結果を記録
      }
    }
    echo implode($result);                                                      # 変更結果(文字+数字)を結合して出力
```
***
 
### 補足  

##### preg_split('/(\d+)/', $target, -1, PREG_SPLIT_DELIM_CAPTURE)
| 引数 | 値 | 意味 |
|:---|:---|:---|
| 第1引数 | `'/(\d+)/'` | 数字1文字以上を`区切りパターン`に指定 |
| 第2引数 | `$target` | 分割`対象の文字列` |
| 第3引数 | `-1` | `分割数`を無制限にする |
| 第4引数 | `PREG_SPLIT_DELIM_CAPTURE` | **区切り文字自体も結果に含める** |

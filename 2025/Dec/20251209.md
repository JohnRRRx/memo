-------------------------------------------
ルールに従う  
・ルールになるA〜Zの英文字が大小ランダムで与えられる  
・大小ランダムの英単語が与えられる  
・与えられた英単語の1文字ずつをルールに沿って大小を変更後、文字列を出力  
  
-------------------------------------------
  
```php
<?php
    $rule = trim(fgets(STDIN));                                       # ルールを取得
    $target = str_split(trim(fgets(STDIN)));                          # 英単語を取得し、1文字ずつ分割
    $result = [];                                                     # 結果を記録用

    foreach ($target as $value){                                      # 各文字を確認していく
      $upper = strtoupper($value);                                    # 大文字を定義
      $lower = strtolower($value);                                    # 小文字を定義
      $result[] = str_contains($rule,$upper) ? $upper : $lower ;      # ルール内に大文字が入ってる場合は大文字、そうでない場合小文字を結果として記録
    }

    echo implode($result);                                            # 変換結果を文字列に結合して出力
```

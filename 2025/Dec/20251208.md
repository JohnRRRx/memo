-------------------------------------------
キーワード検索  
・キーワードが与えられる  
・文章数が与えられる  
・数の分、文章が与えられる  
・各文章に、キーワードが含まれてるかを判定し、入ってる場合はYes、そうでない場合Noと出力  
   
-------------------------------------------
 
```php
                                                           
<?php
    $keyword = trim(fgets(STDIN));                                       # キーワードを取得(trimを使わないと、最後に改行\nが入ってしまったら、あとの処理がうまくいかなくなる)
    $amount = trim(fgets(STDIN));                                        # 文章数を取得
    for ($from = 0; $from < $amount;$from++){                            # 文章数の分、判定していく
      $target = trim(fgets(STDIN));                                      # 文章を取得
      echo str_contains($target,$keyword) ? "Yes"."\n" : "No"."\n";      # キーワードが含まれてる場合Yes、そうでない場合Noと出力
    }
```

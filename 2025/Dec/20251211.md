-------------------------------------------
お得なセール  
・◯点以上購入、◯点が無料  
・全商品数、無料基準商品数、無料商品数が与えられる  
・各商品の値段が与えられる  
・一番安い商品から無料になる  
・全商品を購入する場合の合計金額を出力  
  
-------------------------------------------
  
```
<?php
    list($amounts, $line, $free) = explode(" ",fgets(STDIN));       # 全商品数、無料基準商品数、無料商品数を取得
    $prices = [];                                                   # 各商品の値段を記録用
    for ($i = 0; $i < $amounts; $i++) {                             # 各商品の値段を取得
      $prices[] = trim(fgets(STDIN));
    }
    rsort($prices);                                                 # 商品値段を大〜小に並び替え
   
    if ($amounts >= $line) {                                        # 無料基準に満たした場合
      for ($i = 0; $i < $free; $i++) {                              # 無料商品数の分
        array_pop($prices);                                         # 一番安い商品から0円にする
      }
      echo array_sum($prices);                                      # 合計金額を出力
    } else {                                                        # それ以外の場合
      echo array_sum($prices);                                      # そのまま合計金額を出力
    }
```

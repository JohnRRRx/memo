-------------------------------------------
全員移動  
・部屋数(移動元/移動先)が与えられる  
・それぞれの部屋に入ってる人数が与えられる  
・移動先の部屋が数個あり、それぞれの収容人数が与えられる  
・1部屋対1部屋で、移動元から移動先の部屋に人が入りきれる場合YESと、そうでない場合NOと出力  
  
-------------------------------------------
  
```ruby
before, after = gets.split.map(&:to_i)                     # 部屋数(移動元/移動先)を取得
people = gets.split.map(&:to_i).sort                       # それぞれの部屋にいる人数を取得
rooms = gets.split.map(&:to_i).sort                        # それぞれの移動先の収容人数を取得

room_idx = 0                                               # 部屋のindexを初期化

people.each do |need|                                      # 各移動元の人数を割り当てていく
  while room_idx < rooms.size && rooms[room_idx] < need    # 収容人数が足りない場合
    room_idx += 1                                          # 次の移動先の収容人数を確認
  end
 
  if room_idx >= rooms.size                                # 収容しきれてない場合
    puts "NO"                                              # NOと出力
    exit                                                   # 処理終了
  end
 
  room_idx += 1                                            # この移動先を使ったので次へ
end

puts "YES"                                                 # 処理完了した場合、YESと出力
```

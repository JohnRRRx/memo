Bランク　1問  

-------------------------------------------
🦩と🐢  
・足の総本数、総頭数、1匹の🦩足本数、🐢足本数が与えられる  
・🦩と🐢は少なくとも1匹ずつはいる  
・総本数と頭合計数に沿って、🦩と🐢それぞれ何匹いるかを出力  
・当てはまらない場合、もしくは1通り以上の可能性がある場合、missと出力  
 
-------------------------------------------
 
```ruby
feet, heads, c_feet, t_feet = gets.split.map(&:to_i)                        # 足の総本数、総頭数、1匹の🦩足本数、🐢足本数を取得
cranes = {}                                                                 # 🦩の情報を記録用
tirtles = {}                                                                # 🐢の情報を記録用

(1..heads-1).each_with_index {|head,idx| cranes[idx+1] = head * c_feet}     # 総頭数 - 最低1匹🐢 の全可能性をHashで記録
(1..heads-1).each_with_index {|head,idx| tirtles[idx+1] = head * t_feet}    # 総頭数 - 最低1匹🦩 の全可能性をHashで記録

result = []                                                                 # 結果記録用
cranes.each do |c|                                                          # 🦩と🐢の可能性を交互に確認していく
  c_h = c[0]                                                                # 🦩の数を定義
  c_f = c[1]                                                                # 🦩の足本数を定義
  tirtles.each do |t|
    t_h = t[0]                                                              # 🐢の数を定義
    t_f = t[1]                                                              # 🐢の足本数を定義
    if c_feet == t_feet && heads == 2                                       # 例外：🦩と🐢の足本数が同じ かつ 総頭数2
        result << [1,1].join(" ")                                           # 🦩と🐢1匹ずつの可能性しかないので記録(3匹以上はその3匹目が🦩か🐢のどっちかが分からなくなるため)
    elsif c_h + t_h == heads && c_f + t_f == feet                           # 🦩と🐢の合計数が総頭数と一致し、足合計数と足の総本数も一致した場合
       result << [c_h,t_h].join(" ")                                        # 🦩の数と🐢の数を指定形式で記録
    end
  end
end

puts result.size == 1 ? result : "miss"                                     # 可能性が1つのみの場合🦩の数と🐢の数、それ以外の場合はmissと出力
```

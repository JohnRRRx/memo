Bランク　1問  
 
-------------------------------------------
頭文字の座標を判定  
・正方形の地図の長さ、単語数が与えられる  
・複数の単語が与えられる  
・地図の各座標に英大文字がある  
・↘️方向で読むと単語が現れ、与えられた単語と一致した場合、頭文字の座標のみ出力
 
-------------------------------------------
 
```ruby
len,amounts = gets.split.map(&:to_i)                      # 地図の長さ、単語数を取得
map = len.times.map{gets.chomp.chars}                     # 地図の各座標に英大文字を取得
words = amounts.times.map{gets.chomp.chars}               # 複数の単語を取得

coord = []                                                # 各文字の座標を記録
map.each_with_index { |h,y|                               # 各文字の座標を確認して記録
  h.each_with_index { |w,x|coord << [w,[x+1,y+1]] } }

matched = []                                              # 各単語の頭文字にマッチしたら記録
words.each do |word|                                      # 各単語を確認してく
  initials = word[0]                                      # 頭文字を定義
  steps = word.size - 1                                   # 残りの文字数を定義
  map.flatten.each_with_index{|x,idx|                     # 全地図を一列に並び替える
  matched << [x,idx,steps] if x == initials}              # 頭文字がマッチした場合詳細情報を記録
end

words.each do |origin|                                    # 各単語を確認してく
  matched.each do |first,idx,steps|                       # マッチした頭文字の座標を確認してく
    init_idx = idx                                        # 最後出力用のidx番号を定義
    full = [first]                                        # 全文字を記録用で、頭文字を先に入れとく
      steps.times do                                      # 残りの文字を確認してく
        x_pos = idx % len                                 # それぞれの座標xyを定義
        y_pos = idx / len
        break if x_pos == len - 1 || y_pos == len - 1     # 地図の端っこまで来たら、処理中止
        idx += len+1                                      # idx番号を↘️の文字へ移動
        full << coord[idx][0]                             # その文字を記録
      end
     
      if full == origin                                   # 記録した単語は与えられた単語と一致した場合
        puts coord[init_idx][1].join(" ")                 # 頭文字を規定書式で出力
        break                                             # その単語の処理終了
      end
  end
end
```
